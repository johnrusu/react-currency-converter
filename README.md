# React + TypeScript + Vite: Currency converter

Tehnologies used: React, Typescript, MUI, Tailwind

This is a small, fast currency converter built with React, TypeScript and Vite. It fetches live exchange rates from ExchangeRate-API and lets you convert between major currencies.

Live demo: https://johnrusu.github.io/react-currency-converter/

Quick start

- Install: `npm install`
- Build: `npm run build`
- Serve locally: `npm run dev`

Environment

- The build expects the API key as a Vite environment variable named `VITE_CURRENCY_API_KEY`. For GitHub Pages CI, add a repository secret with the same name.
